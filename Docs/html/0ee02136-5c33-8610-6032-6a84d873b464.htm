      AppCanvas Provides a canvas implementation for drawing graphics using the BOOSE framework. Implements the ICanvas interface to provide drawing operations on a bitmap surface.
      AppCircle Command to draw a circle centered at the current cursor position. Extends CommandOneParameter to handle the radius parameter.
      AppCommandFactory Custom command factory for the BOOSE application that creates command instances. Extends the base CommandFactory to provide custom implementations of drawing commands.
      AppMoveTo Command to move the drawing cursor to a specified position without drawing a line. Extends CommandTwoParameters to handle X and Y coordinates as parameters.
      AppRect Command to draw a rectangle at the current cursor position. Extends CommandOneParameter but handles two parameters for width and height.
      Form1 Main form for the BOOSE application, providing a GUI for executing BOOSE programs and displaying canvas output.
